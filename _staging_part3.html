Network Effect

2 ฝ่งผู้ตรวจสอบเอกสาร (Verifiers)

กลุ่มขยายฐานเพื่อดึงดูดผู้ใช้งานเข้าสู่ระบบ

🎯 Target Group (TH): ฝ่ายบุคคล (HR), สถาบันการเงินที่ต้องทำ Credit Scoring/KYC, แพลตฟอร์ม Onboarding
⚠️ Deep-Dive Pain Point:

VDR ในไทยกระจัดกระจาย (Fragmentation) ฝ่งคนตรวจไม่อยากเขียนโค้ดเพื่อไปต่อ API กับ VDR หลายๆ ที่เพื่อตรวจเช็กสถานะการ Revoke (เพิกถอน) ต้องการโซลูชันเดียวที่ฟันธงได้เลย

💡 SaaS Opportunity: "1-Click Verification Portal"

แนวทางโปรดกส์: สร้าง Registry Aggregator รวมทุก VDR ไว้ที่เดียว เมื่อ Verifier สแกน VC ระบบจะวิ่งไปตรวจ Cryptographic Proof, DID, และ Revocation List ทันที พร้อมออก Audit Report

💸 Monetization: Monthly/Annual SaaS Subscription (Seat-based) หรือ Pay-per-Verification API call

High Barrier to Entry

3 ผู้ให้บริการโครงสร้างข้อมูล (VDR)

กลุ่มผู้สร้างรากฐานระดับอุตสาหกรรม

🎯 Target Group (TH): สมาคมวิชาชีพ, Consortium ของกลุ่มธุรกิจ, หรืออุตสาหกรรมเฉพาะทางที่ต้องการสร้าง Trust Network
⚠️ Deep-Dive Pain Point:

การตั้ง Node VDR ให้มี Uptime 99.99% ตลอดเวลา และระบบการจัดการ Revocation List (สถานะเอกสารถูกยกเลิก) แบบ Low Latency มีต้นทุน Infrastructure และ Operation ที่สูงและซับซ้อนมาก

💡 SaaS Opportunity: "Managed VDR / Revocation Node"

แนวทางโปรดกส์: ระบบคล้าย AWS แต่เจาะจงเฉพาะ SSI โดยให้บริการเช่าพื้นที่ VDR Node ที่ปลอดภัยสูง จัดการระบบ Revocation ทันใจ (เช่น Cryptographic Accumulators)

💸 Monetization: Compute & Storage Resource Subscription + SLA Contracts ระดับองค์กร

B2B2C Interface

4 กระเป๋าเอกสารผู้ใช้งาน (Wallet Providers)

จุดเชื่อมต่อประสบการณ์ประชาชนทั่วไป

🎯 Target Group (TH): นิติบุคคลโครงการอสังหาฑฯ, องค์กรที่มีสมาชิกลูกค้า (Membership)
⚠️ Deep-Dive Pain Point:

การบังคับประชาชนโหลดแอพพลิเคชัน (App) ใหม่เพื่อเก็บ VC เป็นอุปสรรคสำคัญ (Friction) ในขณะที่ธุรกิจก็ไม่อยากลงทุนพัฒนาแอพพลิเคชัน Wallet ที่ปลอดภัยสูงขึ้นมาเอง

💡 SaaS Opportunity: "White-Label Wallet SDK / LINE Mini App"

แนวทางโปรดกส์: พัฒนา Wallet เป็น SDK ฝังเข้าแอพพลิเคชันเดิมของลูกค้า หรือใช้แพลตฟอร์มยอดนิยมอย่างสร้าง LINE Mini App ให้ผู้ใช้เปิดรับและเก็บ VC ได้ทันที

💸 Monetization: B2B Licensing & Setup Fee (องค์กรเป็นผู้จ่าย) + ค่า MAU (Monthly Active Users)
